# 데이터베이스 명세서(ERD)

- **테이블 예시**
  - **users**
    - id (PK, int, auto)
    - username (varchar, unique)
    - password (varchar)
    - role (varchar)
    - created_at (datetime)
  - **buses**
    - id (PK, int, auto)
    - bus_number (varchar, unique)
    - status (varchar)
    - ...
  - **drivers**
    - id (PK, int, auto)
    - name (varchar)
    - phone (varchar)
    - ...
  - **schedules**
    - id (PK, int, auto)
    - bus_id (FK, int)
    - driver_id (FK, int)
    - departure_time (datetime)
    - ...
  - **notifications**
    - id (PK, int, auto)
    - user_id (FK, int)
    - message (text)
    - is_read (boolean)
    - created_at (datetime)
- **관계**
  - users 1:N notifications
  - buses 1:N schedules
  - drivers 1:N schedules
  - schedules N:1 buses, N:1 drivers
