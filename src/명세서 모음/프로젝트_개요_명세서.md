# 프로젝트 개요 명세서

- **프로젝트 이름:** (예: 스마트 버스 관리 시스템)
- **목적/한 줄 설명:** 버스 운행, 기사, 스케줄, 알림 등을 통합 관리하는 대시보드형 웹 서비스
- **해결하려는 문제:** 버스 운행 현황, 기사 배차, 실시간 알림 등 교통 운영의 비효율성 및 정보 단절 해소
- **주요 기능 목록:**
  - 버스/기사/스케줄 CRUD
  - 실시간 알림 및 알림 위젯
  - 대시보드(운행 현황, 통계 차트)
  - 사용자 인증/프로필 관리
  - WebSocket 기반 실시간 데이터 반영
- **대상 사용자:** 교통 운영 관리자, 버스 기사, 시스템 운영자
- **현재 진행 상황:** 프론트엔드 개발 중(React), 일부 API 연동 및 UI 구현 완료
